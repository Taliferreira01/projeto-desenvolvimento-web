Minha Formação Acadêmica

Atualmente, estou cursando o curso de Tecnologia em Banco de Dados na Uninter, finalizando o 2° semestre. Estou desenvolvendo habilidades essenciais para o manuseio e a administração de grandes volumes de dados, além de me aprofundar em sistemas de gerenciamento de banco de dados (SGBDs) e técnicas de modelagem de dados.

Durante o curso, tenho aprendido sobre:

  • Estruturas de Dados - Como organizar e otimizar o armazenamento de dados.
  • SQL e NoSQL - Domínio das linguagens de consulta, como SQL, e sistemas de banco de dados NoSQL.
  • Modelagem de Dados - Técnicas de criação de modelos de dados relacionais e não relacionais.
  • Administração de Bancos de Dados - Gerenciamento de segurança, backups e recuperação de dados.
  • Performance e Otimização - Estratégias para melhorar o desempenho de consultas e manipulação de dados.

Estou muito motivada para aplicar esses conhecimentos em projetos reais e continuar me desenvolvendo na área de Banco de Dados.



Objetivos Profissionais

Meu objetivo profissional é me tornar especialista em Administração de Banco de Dados e atuar no gerenciamento de sistemas de dados complexos, com foco em otimização, segurança e integridade das informações. Tenho interesse em trabalhar com empresas que lidam com grandes volumes de dados e que utilizam tecnologias avançadas de armazenamento e processamento.
